# Взвешенная система оценивания тестов

## Общая модель

В системе применяется взвешенное оценивание, при котором для каждого вопроса рассчитываются:

- `r_i` — коэффициент оценивания ответа на вопрос
- `z_i` — уровень сложности вопроса

Итоговый процент прохождения теста вычисляется по формуле:

`R = (Σ(r_i * z_i) / Σ z_i) * 100`

где:

- `0 <= r_i <= 1`
- `z_i ∈ {1, 2, 3}`
- `R` — итоговый процент результата по тесту

## Уровни сложности

- `z_i = 1` — базовый вопрос
- `z_i = 2` — вопрос средней сложности
- `z_i = 3` — вопрос повышенной сложности

Если для вопроса сложность явно не задана, используется значение `1`.

## Формулы по типам вопросов

### SINGLE_CHOICE

Для вопроса с одним правильным вариантом:

- `r_i = 1`, если выбран правильный ответ
- `r_i = 0`, если выбран неправильный ответ или ответ отсутствует

### MULTIPLE_CHOICE

Для вопроса с несколькими правильными вариантами применяется мягкое частичное оценивание:

`r_i = max(0, correctSelected - wrongSelected) / correctTotal`

где:

- `correctSelected` — количество правильно выбранных вариантов
- `wrongSelected` — количество ошибочно выбранных вариантов
- `correctTotal` — общее число правильных вариантов в вопросе

Если пользователь отметил только неправильные варианты, коэффициент принимается равным `0`.

### MATCHING

Для вопроса на установление соответствия:

`r_i = correctlyMatchedPairs / totalPairs`

где:

- `correctlyMatchedPairs` — число верно составленных пар
- `totalPairs` — общее количество пар в вопросе

### SEQUENCE

Для вопроса на установление правильной последовательности:

`r_i = correctlyPositionedItems / totalItems`

где:

- `correctlyPositionedItems` — количество элементов, стоящих на правильных позициях
- `totalItems` — общее число элементов последовательности

### OPEN_TEXT

Для открытого вопроса применяется автоматическая дихотомическая оценка:

- `r_i = 1`, если ответ совпадает с эталоном без учета регистра и лишних пробелов по краям
- `r_i = 0`, если ответ не совпадает или отсутствует

## Пример расчета

Пусть тест состоит из трех вопросов:

1. Вопрос 1: `r_1 = 1`, `z_1 = 1`
2. Вопрос 2: `r_2 = 0.5`, `z_2 = 2`
3. Вопрос 3: `r_3 = 0`, `z_3 = 3`

Тогда:

- `Σ(r_i * z_i) = 1 * 1 + 0.5 * 2 + 0 * 3 = 2`
- `Σ z_i = 1 + 2 + 3 = 6`
- `R = (2 / 6) * 100 = 33.3%`

## Отображение в интерфейсе

В результатах тестирования для каждого вопроса отображаются:

- тип вопроса
- уровень сложности
- коэффициент ответа `r_i`
- статус зачета:
  - `Полный зачет`, если `r_i = 1`
  - `Частичный зачет`, если `0 < r_i < 1`
  - `Без зачета`, если `r_i = 0`

Итоговый процент на странице результата и в Excel-отчете рассчитывается именно по взвешенной формуле, а не как простая доля полностью верных ответов.
